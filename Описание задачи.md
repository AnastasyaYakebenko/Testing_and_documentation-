## Проект: Тестирование сайта бронирования отелей

**Задача:** Проведение комплексного тестирования сайта (регистрация, личный кабинет, бронирование отелей и управление бронированиями) для определения готовности к передаче пользователям. Подготовка полного комплекта документации для заказчика.

### Выполненные работы:

**1. Функциональное тестирование**
Провела ручное тестирование ключевого функционала сайта:
- Регистрация и авторизация пользователей
- Личный кабинет
- Бронирование отелей
- Управление бронированиями (изменение, отмена, просмотр)

**2. Планирование тестирования**
Составила план тестирования с подробными тест-кейсами, провела тестирование сайта в соответствии с утвержденным планом.

**[План тестирования с тест-кейсами (Google Docs)](https://docs.google.com/document/d/1OsYer1JBXmykVIEQG8q1vr0K-piLbFyl83vNv9wg0mI/edit?usp=sharing)**

**3. Разработка документации**

| Документ | Назначение | Ссылка |
|----------|------------|--------|
| **Программа и методика испытаний (ПМИ)** | Документ, регламентирующий порядок, объем, условия и методы проведения испытаний сайта | [Открыть ПМИ](https://docs.google.com/document/d/1-uKgT2Y6S69j-vuXQVsVysO9StZUrZbmiHMR6R5m-s4/edit?usp=drive_link) |
| **Руководство пользователя** | Инструкция по работе с сайтом для конечных пользователей | [Открыть руководство](https://docs.google.com/document/d/16fcZN3FoZFTBr9DZtfOZIGythdb6noB1wCgekC3EMMk/edit?usp=sharing) |

Все документы подготовлены для проверки заказчиком соответствия выполненных работ предъявленным требованиям.

### Ключевые навыки, продемонстрированные в проекте:
- Ручное тестирование веб-приложений
- Составление тест-планов и тест-кейсов
- Разработка программы и методики испытаний (ПМИ)
- Создание пользовательской документации
- Подготовка отчетной документации для заказчика
